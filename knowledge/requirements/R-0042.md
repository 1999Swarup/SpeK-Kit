# Users can reset their password via a time-limited email link

## Statement
A user can request a password reset; the system emails a single-use link that
expires after a bounded time and lets them set a new password.

## Acceptance criteria
- [ ] Given a valid account email, when reset is requested, then a single-use
      link is emailed.
- [ ] Given a reset link, when it is older than the expiry window, then it is
      rejected.
- [ ] After N failed attempts the account is temporarily locked. *(depends on Q-0005)*

## Rationale
Closes the top support-ticket category and unblocks locked-out users.

- [[Q-0005]] — what are N and the lockout duration?
