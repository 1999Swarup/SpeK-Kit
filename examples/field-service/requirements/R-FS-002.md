# Field technicians can view and update work orders offline, syncing on reconnect

## Statement
The Field Service mobile app (Power Apps) provides an offline profile so
technicians can read and edit assigned work orders, capture notes, parts, and
signatures without connectivity; changes queue and sync when online.

## Acceptance criteria
- [ ] Given no network, when a technician opens the app, then assigned work
      orders and related records are available from the offline store.
- [ ] Given edits made offline, when connectivity returns, then changes sync and
      any conflict is resolved by the defined policy (no silent data loss).
- [ ] Sensitive customer data at rest on device is encrypted.

## Rationale
Field work happens where signal does not.
